# WallMobi — Mobile AI Wallpaper Gallery

A production-ready **Next.js 16** website for downloading original, AI-generated **phone**
wallpapers. Clean, Apple-style light UI, mobile-first portrait artwork, and a fully
localized interface in **6 languages**.

## Languages

| Code | Language | UI status |
|------|----------|-----------|
| `en` | English | reference |
| `th` | ไทย (Thai) | complete |
| `vi` | Tiếng Việt (Vietnamese) | complete |
| `my` | မြန်မာ (Burmese) | machine-assisted — **have a native speaker review** |
| `lo` | ລາວ (Lao) | machine-assisted — **have a native speaker review** |
| `km` | ខ្មែរ (Khmer) | machine-assisted — **have a native speaker review** |

> The Burmese, Lao and Khmer strings are careful best-effort translations. Short UI
> labels are reliable; longer marketing sentences should be proofread by a native
> speaker before launch. All copy lives in `src/i18n/dictionaries/*.ts`.

## Stack & key features

- Next.js 16 (App Router) · React 19 · TypeScript
- `/[locale]` routing for all pages, 145 fully static pre-rendered pages
- SEO: per-page metadata, **hreflang** alternates + `x-default`, canonical URLs,
  localized `sitemap.xml`, `robots.txt`, JSON-LD (WebSite, Organization, SearchAction,
  ImageObject, BreadcrumbList), dynamic OpenGraph image, web manifest, app icon
- Client-side **search** + category filter + "load more"
- One-tap **download** with a localized button
- **Self-hosted fonts** (Noto Sans family) covering Latin, Vietnamese, Thai, Myanmar,
  Lao and Khmer via a single `AuraSans` family scoped with `unicode-range`

## Run it

```bash
npm install
npm run dev      # http://localhost:3000  → redirects to /en (or your browser language)
npm run build    # production build
npm start        # serve the build
```

## Before you deploy

1. **Set your domain** in `src/lib/site.ts` → `site.url` (no trailing slash). Everything
   SEO-related (canonical, hreflang, sitemap, JSON-LD) is built from it.
2. **Swap in real wallpapers.** The 12 files in `public/wallpapers/*.svg` are
   placeholders generated by `scripts/generate-wallpapers.cjs`. Replace them with your
   own AI-generated **portrait** images (1080×2340 or similar) and update the list in
   `src/lib/wallpapers.ts` (slug, title, category, localized `desc`, dimensions, tags).
   If you host images on a CDN, add the host to `images.remotePatterns` in
   `next.config.ts`.
3. **Review the my / lo / km copy** with native speakers (see table above).

## Editing translations

- UI strings: `src/i18n/dictionaries/<locale>.ts` (each file is typed against `en.ts`,
  so missing/renamed keys fail the build — a safety net).
- Add a language: add the code to `src/i18n/config.ts`, create the dictionary file,
  register it in `src/i18n/index.ts`, and (if it's a new script) add a `@font-face`
  with the right `unicode-range` in `src/app/globals.css` plus the woff2 in
  `public/fonts/`.

## Fonts

Google Fonts is fetched at build time by `next/font/google`; this project instead
**self-hosts** woff2 subsets (better privacy, no build-time network dependency, works
offline). They're combined into one `AuraSans` family in `globals.css`, and the browser
picks the correct script automatically via `unicode-range`.

## A note on AI images & licensing

AI-generated images generally **cannot be exclusively copyrighted**, and commercial use
depends on the terms of the generator you used. Avoid prompts that reproduce trademarks,
fictional characters, or real people. The on-site License page explains usage to visitors;
adjust it to match your own terms.
